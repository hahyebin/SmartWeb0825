--- 상속(INHERIT)

1. 어떤 클래스의 기능을 다른 클래스에게 상속해 주는 것이다.
2. 부모 클래스의 기능을 자식 클래스에게 상속해 주는 것이다.

3. 상속이 가능한 관계
  1) is - a : 자식 is a 부모
  2) has - a : 자식 has a 부모
  
 4. super : 부모 클래스
 5. sub   : 자식 클래스
 
 
 
 
 * Object (조상 클래스)
  : 명시하지 않아도 모든 클래스는 object 클래스를 상속 
  
  
 * 자바는 다중상속이 안됨 : 조상 메서드들끼리 충돌하기 때문에 금지함
